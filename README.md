## ✨ Features

### 🌿 Core Features
- **Habit Tracking**: Create, manage, and track daily habits with ease
- **Garden Visualization**: Your virtual garden grows and evolves based on habit completion
- **Progress Dashboard**: Visual progress indicators and statistics at a glance
- **Streak Counter**: Track your consistency with streak monitoring
- **Daily Journal**: Record thoughts, reflections, and track your mood

### 🎨 User Experience
- **Beautiful Material 3 Design**: Modern, clean interface following Material Design 3 guidelines
- **Responsive Layout**: Optimized for all screen sizes - mobile, tablet, and desktop
- **Dark/Light Theme Support**: Comfortable viewing in any lighting condition
- **Smooth Animations**: Delightful micro-interactions and transitions
- **Intuitive Navigation**: Easy-to-use bottom navigation bar

### 🏆 Gamification Elements
- **Achievement System**: Unlock achievements as you build consistency
- **Level Progression**: Advance through garden levels
- **Plant Collection**: Collect various plants as rewards
- **Seasonal Themes**: Garden appearance changes with seasons
