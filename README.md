 #  Habit Gardenia

Transform your daily routines into an exciting journey of growth! HabitGardenia combines habit tracking with a digital notebook in a gamified experience where consistency helps your virtual garden flourish.

##  Overview

Habit Gardenia is a mobile application designed to make habit-building and daily planning engaging and fun. Users can manage their daily activities, maintain a personal journal, and watch their progress manifest through an interactive garden ecosystem - complete with growing trees for completed tasks and falling leaves for missed activities.

##  Key Features

###  Activity Management
- Create and customize daily activities and routines
- Set reminders and schedule recurring tasks
- Track completion progress with intuitive checklists
- Categorize activities for better organization

###  Digital Notebook
- Rich text notes with formatting options
- Organize thoughts, ideas, and reflections
- Link notes to specific activities or dates
- Search and filter through journal entries

###  Gamification Elements
- **Growing Garden**: Watch trees flourish as you complete activities
- **Visual Feedback**: Leaves fall when tasks are missed, creating an engaging consequence system
- **Progress Unlocks**: Earn new plant species and garden decorations through consistent habit-building
- **Achievement System**: Unlock badges for reaching milestones

###  User Experience
- Clean, intuitive Material Design interface
- Smooth animations and transitions
- Customizable themes and garden aesthetics
- Dark/Light mode support

##  Technical Stack

- **Framework**: Flutter
- **Language**: Dart
- **Architecture**: BLoC / Riverpod for state management
- **UI**: Flutter Widgets with custom animations
- **Database**: sqflite / Hive / Drift for local storage
- **Dependency Injection**: get_it / injectable
- **Background Processing**: flutter_local_notifications / workmanager
- **State Management**: BLoC / Riverpod / Provider
